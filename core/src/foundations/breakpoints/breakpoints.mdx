## Usage

Load Fudis mixins and tokens to your SCSS file with:

```
@use "@funidata/fudis-core" as core;
```

### Min-width

When you are using `breakpoint('md')`, your styles will be applied from `$breakpoint-md` and up.

```
@include core.breakpoint($breakpoint){}
```

<table>
    <thead>
    <tr>
        <th>$breakpoint</th>
    </tr>
    </thead>

    <tbody>
    <tr>
        <td>string value from breakpoint tokens map (**$breakpoints**)</td>
    </tr>
    </tbody>

```
.my-element {
  padding-top: core.$spacing-md;
  @include core.breakpoint('md') {
    // Rules defined here apply from breakpoint 'md' and up.
    padding-top: core.$spacing-lg;
  };
}
```

### Max-width

When you are using `breakpoint-max('md')`, your styles will be applied when viewport is smaller than `$breakpoint-md`.

```
@include core.breakpoint-max($breakpoint){}
```

<table>
    <thead>
    <tr>
        <th>$breakpoint</th>
    </tr>
    </thead>

    <tbody>
    <tr>
        <td>string value from breakpoint tokens map (**$breakpoints**)</td>

```
.my-element {
  padding-top: core.$spacing-lg;
  @include core.breakpoint-max('md') {
    // Rules defined here apply up to breakpoint 'md'.
    padding-top: core.$spacing-sm;
  };
}
```
