## Tokens

There are two supported border styles, defined below:

<table>
    <thead>
    <tr>
        <th>$border-styles</th>
    </tr>
    </thead>

    <tbody>
    <tr>
        <td>solid</td>
    </tr>

    <tr>
        <td>dashed</td>
    </tr>

    <tr>
        <td>none</td>
    </tr>

    </tbody>

</table>

## Border

Set border of an element.

```
@include core.border($width, $style, $color)
```

<table>
    <thead>
    <tr>
        <th>$width</th>
        <th>$style</th>
        <th>$color</th>
    </tr>
    </thead>

    <tbody>
    <tr>
        <td>string value from spacing tokens map (**$pixel-tokens**)</td>
        <td>string value from border tokens map (**$border-styles**)</td>
        <td>string value from colors tokens map (**$colors**)</td>
    </tr>
    </tbody>

</table>

### Example

Load Core mixins and tokens to your SCSS file with:

```
@use "@funidata/fudis-core" as core;
```

```
.my-element {
  @include core.border('2px', 'solid', 'primary');
}
```

## Border None

Remove or set the border to none. Border none doesn't need any parameters.

Load Core mixins and tokens to your SCSS file with:

```
@use "@funidata/fudis-core" as core;
```

```
.my-element {
  @include core.border-none;
}
```
