### Usage

Load Core mixins and tokens to your SCSS file with:

```
@use "@funidata/fudis-core" as core;
```

```
.grid-container {
  // set a max-width to grid container by using a token
  max-width: core.$grid-width-lg;

  gap: core.$spacing-xxs;
  display: grid;
  grid-template-columns: auto auto auto auto;
}
```

## Max-width mixin

Load Core mixins and tokens to your SCSS file with:

```
@use "@funidata/fudis-core" as core;
```

Mixin will set selected element's CSS max-width to given Grid token width

```
.my-element {
  @include fudis.grid-max-width('lg');
}
```
