### Usage

When using headings with mixins, load core to your SCSS file:

```
@use "@funidata/fudis-core" as core;
```

The mixins for headings are:

- heading-xxl
- heading-xl
- heading-lg
- heading-md
- heading-sm
- heading-xs
- heading-xxs

#### Example

```
.my-element {
  @include core.heading-xxl;
}
```

## Table caption

Table caption is meant to be used with the corresponding `caption` HTML-element to provide the table an accessible description. You can implement this with a mixin that is documented on this page or with a class that is documented in the [Miscellaneous-section](/story/components-typography-miscellaneous--table-caption).

<Typeset
  fontSizes={[Number(body.size.md)]}
  fontWeight={Number(body.weight.regular)}
  sampleText={
    <p style={{ textTransform: "uppercase" }}>Fudis table caption text</p>
  }
  fontFamily={body.type.primary}
/>

### Usage

When using the table caption, load core to your SCSS file:

```
@use "@funidata/fudis-core" as core;
```

Include the mixin to your class:

```
.my-element {
  @include core.table-caption;
}
```

## Empty state

Empty state is used to express that there is no information available. You can implement this with a mixin that is documented on this page or with a class that is documented in the [Miscellaneous-section](/story/components-typography-miscellaneous--empty-state).

<Typeset
  fontSizes={[Number(body.size.md)]}
  fontWeight={Number(body.weight.regular)}
  sampleText={
    <p style={{ fontStyle: "italic", color: "#727272" }}>
      Fudis empty state text
    </p>
  }
  fontFamily={body.type.primary}
/>

### Usage

When using the empty-state, load core to your SCSS file:

```
@use "@funidata/fudis-core" as core;
```

Include the mixin to your class:

```
.my-element {
  @include core.text-emptystate;
}
```
