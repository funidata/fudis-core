Primary colors are defined in the application stylesheet, using the predefined variables documented in the [Define Application's Root Styles]()-section. In this section there are presented some default primary color-palettes, that are used when developing an application that has a need for the Sisu or Into color palettes. You can also define your own primary colors, but the seccondary and accent colors are defaulted. All colors can be accessed in the using application by using tokens.

## Color tokens

You can access the colours in your application stylesheet when needed by using tokens.

Start by loading the core assets to your stylesheet:

```
@use "@funidata/fudis-core" as core;
```

Then you can start using the tokens:

```
.my-element {
  background-color: core.$color-primary-extra-dark;
}
```
